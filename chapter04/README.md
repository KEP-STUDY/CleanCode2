# 주석

## 대 전제

- 코드는 계속 변화하고 진화한다.
  - 주석은 코드를 수정할때 수정해야하는 포인트이지만 놓치기 쉽다.
- 주석은 문제를 해결할 수 없다.
- 주석이 틀릴때 오류를 유발할 수 있다.
- 코드로 의도를 표현하라!
  - 우리는 프로그래머. 코드로 말하는 것이 직관적이고 위험하지 않다
